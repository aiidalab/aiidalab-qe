Tasks

Anti-ferromagnetic calculations require different types of a certain element, to define the spin-up and spin-down initialization of a given chemical element. Depending on the size of the unit cell (even number of sites of the element of interest), it might be necessary to create a supercell first.
  1. Click on the From examples tab
  2. Click on Simple crystals
  3. Select Cobalt oxide from the dropdown list
  4. Click on the Edit structure panel and open the Edit cell tab. Set the matrix element (3, 3) and click Apply transformation to 2 to double the unit cell.
  5. Switch to the Edit atom tags tab.
  6. Select the site at the corner of the unit cell that we will define as spin-down in the structure visualizer above. In the Edit atom tags tab, press the From Selection button and select the tag 1. Press the Update tags button to apply the changes.
  7. Repeat the previous step for spin-up, but this time, select the other Co atom and set the tag to 2.
  8. Click on the Confirm button to proceed

In the submission step, we define the computational resources to be used in the calculations. The global resources are used to define resources across all workflow calculations. Optionally, you can override the resource settings for specific calculations.

Warning: If running locally (for example, on the AiiDAlab demo server), we recommend keeping nodes and CPUs at the default minimum of 1 each.

Post-guide exercises

  1. Use the app to compute the anti-ferromagnetic band structure.
    • Follow the same steps as before, but this time, select the electronic band structure in the Configure workflow panel.
    • You can also check both boxes, to calculate the PDOS and band structure. This will allow you to plot so called fat-bands, showing the orbital character of the bands.
  2. Compute the ferromagnetic PDOS.
    • To have the same number of atoms per cell, you should consider to use the same supercell as before. However, you do not need to set the tags.
    • Set the initial magnetic moments to the same values as before.
    • Finally, compare your FM PDOS with the AFM one obtained before.
